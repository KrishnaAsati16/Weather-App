
# 🌤️ Weather App

A beautiful real-time weather app built with React & TypeScript.

## ✨ Features

- 📍 Auto-detects your current location
- 🔍 Search weather by city name
- 🌡️ Shows temperature, humidity & wind speed
- 🎨 Dynamic weather icons based on conditions

## 🛠️ Tech Stack

- React + TypeScript
- Styled Components
- OpenWeatherMap API
- Axios

## 🚀 Getting Started

### Installation

```bash
git clone https://github.com/your-username/weather-app.git
cd weather-app
npm install
```

### Run the App

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

## 🔑 API Key Setup

1. Go to [openweathermap.org](https://openweathermap.org) and create a free account
2. Copy your API key
3. Paste it in `DisplayWeather.tsx`:
```ts
   const api_key = "YOUR_API_KEY";
```
